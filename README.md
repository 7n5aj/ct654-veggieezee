**Veggieeze** is an intelligent vegetable price prediction system designed to help farmers, traders, and agricultural businesses make data-driven decisions. By leveraging machine learning algorithms and real-time market data, Veggieeze provides accurate price forecasts, seasonal insights, and personalized recommendations.


##  How to run the project? 
## ‚úÖ Prerequisites
   Before setting up the project, make sure the following are installed on your system:
1. **Python 3.9+**
   Check installation: 
   ```bash
   python --version
   ```
2. **uv(Python Package Manager)**
    Install uv if not already installed: 
   ```bash
   pip install uv
   ```
   Verify installation:
   ```bash
   uv --version
   ```

   
## üöÄ Get Started
    Follow these steps to get the project up and running:
1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/ct654-veggieezee.git
   cd ct654-veggieezee
   ```

2. **Create a Virtual Environment**
   ```bash
   uv venv
   ```

3. **Activate the Virtual Environment**
   ```bash
   .venv\Scripts\activate
   ```

4. **Install Dependencies**
   ```bash
   uv pip install -r requirements.txt
   ```

5. **Database Migrations**
   ```bash
   python manage.py migrate
   ```

6. **Run Development Server**
   ```bash
   python manage.py runserver
   ```

## üíª Usage

### For New Users

1. **Register**: Create an account with your farm details
2. **Select Crop**: Choose your primary vegetable crop
3. **Get Predictions**: Input location and get price forecasts
4. **Analyze Insights**: View trends, charts, and recommendations

### For Returning Users

1. **Login**: Access your personalized dashboard
2. **View Profile**: Check your farm information and preferences
3. **Track Prices**: Monitor real-time price updates
4. **Receive Alerts**: Get notifications on significant price changes
